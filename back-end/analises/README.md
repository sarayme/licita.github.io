# Analisando os dados de exonerações e nomeações

## Requisitos

```sh
python3 -m venv .venv
source .venv/bin/activate
pip3 install -r requirements.txt
```

Depois disso, basta abrir o vscode. Precisa ter a extensão do jupyter instalada.
